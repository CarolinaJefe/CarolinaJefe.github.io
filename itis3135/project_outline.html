Project Outline

Project Overview

  • This application aims to enhance the organization's visibility and strengthen community engagement. Its primary function is to raise awareness about the organization while providing a platform for acknowledging achievements and responsibilities.
  • The intended users of the website are generally everyone, as it's used as a means of recognition and embodiment. However, specifically, those within the organization are the primary focus.
  • The content within the website will include, but is not limited to, organization information, member information, organization timeline, contact information, organization accomplishments, and organization signup information.

Page Design

Home

  • The purpose of this page is to welcome users and describe the organization's who, what, when, where, why, and how.
  • The audience targeted are both newcomers and members.
  • The page will contain various sections that revolve around explaining the organization's overall purpose. Additionally, there may be a static events/feed on the page.
  • This page does not ask users to enter data on the page.
  • The page will possibly contain hyperlinks for specific keywords and events/feed tab.
  • No actions will occur other than URL redirection.

Members

  • The purpose of this page is to welcome users about the members involved within the organization.
  • The audience targeted are both newcomers and members.
  • The page will contain profile cards for each member involved within the organization.
  • This page may ask users to enter data on the page for filtering.
  • Data sanitization is expected as you should never trust data coming in from the user.
  • The page will contain buttons, hyperlinks, and dropdowns for viewing the profile and contacting said member, if applicable.
  • As previously mentioned, actions will include browsing the profile and contact information for the respective member.

Invasions

  • The purpose of this page is to welcome users about the organization's accomplishments and current progress in wars.
  • The audience targeted are specifically newcomers.
  • The page will contain various sections that revolve around explaining the organization's achievements and current progression.
  • This page does not ask users to enter data on the page.
  • The page will possibly contain hyperlinks and media for certain achievements.
  • No actions will occur other than URL redirection and media usage.

Law & Order

  • The purpose of this page is to welcome users about the organization's rules.
  • The audience targeted are specifically newcomers.
  • The page will contain various sections that revolve around explaining the organization's rules and expectations.
  • This page does not ask users to enter data on the page.
  • The page will not contain external media.
  • No actions will occur other than URL redirection.

Contact

  • The purpose of this page is to welcome users about the various ways to contact the organization.
  • The audience targeted are specifically newcomers.
  • The page will contain links for contacting a representative of the organization.
  • This page should not ask users to enter data on the page for contacting. This is for simplicity's sake.
  • The page will contain buttons and hyperlinks for contacting a representative.
  • As previously mentioned, actions will include sending an email or being redirected to an appropriate method of contact.
